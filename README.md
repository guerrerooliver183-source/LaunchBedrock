# Bedrock Launcher

Bedrock Launcher es una aplicación Android diseñada para mejorar la experiencia de inicio de Minecraft Trial, ofreciendo una interfaz moderna basada en Material You y funcionalidades avanzadas de personalización.

## Características Principales

*   **Diseño Material You:** Adaptación dinámica a los colores del sistema y soporte completo para modo claro y oscuro.
*   **Secuencia de Inicio:** Pantalla de carga profesional con barra de progreso segmentada y textos de estado.
*   **Gestión de Actualizaciones:** Sistema de notificaciones para nuevas versiones con changelog detallado e instalación directa.
*   **Ajustes Personalizados:** Control total sobre el tema, el inicio automático y la visibilidad de la aplicación original.
*   **Seguridad:** Uso de permisos de administrador del dispositivo para ocultar o mostrar la aplicación de Minecraft Trial de forma segura.

## Requisitos de Desarrollo

*   **Android Studio:** Recomendado para la compilación y edición del código.
*   **Java:** Lenguaje principal utilizado para la lógica de la aplicación.
*   **Gradle:** Sistema de construcción del proyecto.

## Instalación y Uso

1.  Clona este repositorio en tu máquina local.
2.  Abre el proyecto en Android Studio.
3.  Asegúrate de tener instalado Minecraft Trial en tu dispositivo para un funcionamiento óptimo.
4.  Compila y ejecuta la aplicación en un emulador o dispositivo físico.

## Estructura del Proyecto

El proyecto sigue una estructura organizada en paquetes para facilitar su mantenimiento:

*   `com.minecraft.launcher`: Contiene las actividades principales y el receptor de administrador.
*   `com.minecraft.launcher.utils`: Utilidades para seguridad, detección de paquetes y verificación de actualizaciones.
*   `res/layout`: Definiciones de la interfaz de usuario en XML.
*   `res/values`: Recursos de color, temas, textos y dimensiones.
