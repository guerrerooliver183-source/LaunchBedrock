# Versiones de Bedrock Launcher

## v1.0.0 (2026-05-26)
- Lanzamiento inicial del proyecto.
- Implementación de diseño Material You con soporte para modo claro y oscuro.
- Pantalla de carga con secuencia de estados: Checking for updates, Starting Minecraft, Injecting.
- Barra de progreso segmentada en 4 secciones.
- Sistema de notificaciones integradas para actualizaciones.
- Pantalla de ajustes con opciones generales y de gameplay.
- Detección automática de Minecraft Trial.
- Funcionalidad para ocultar/mostrar la aplicación mediante administrador de dispositivos.
- Cambio automático a orientación horizontal al finalizar la carga.
- Implementado en Java con layouts XML para máxima compatibilidad con Android Studio.
